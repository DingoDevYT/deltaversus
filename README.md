# DELTAVERSUS

A DELTARUNE fan game: 1v1 turn-based bullet-hell PvP. Pick a fighter
(Kris, Susie, Ralsei, Noelle, Lancer), choose your attacks, then dodge
your opponent's patterns in real time. Built with vanilla JS + canvas,
multiplayer over WebRTC (PeerJS) with 4-letter room codes.

**Play:** open the GitHub Pages URL, one player hosts, the other joins
with the room code. Or PRACTICE vs a dummy.

Controls: Arrows/WASD move - Z/Enter confirm - X/Shift cancel.
